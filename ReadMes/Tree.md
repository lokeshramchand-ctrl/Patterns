# Trees Practice Roadmap (Up to ~10 LPA Interview Level)

This checklist covers the essential Binary Tree and Binary Search Tree problems typically asked in coding interviews for companies in the ~8–10 LPA range.

Goal:
- Understand recursion on trees
- Master traversal patterns
- Learn basic BST operations
- Solve common interview-style tree problems
